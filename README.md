# Employee Salary Calculator.xlsx

## Project Overview

This project calculates employee bonuses, total salary, and salary status using Microsoft Excel formulas.

## Features

* Bonus Calculation
* Total Salary Calculation
* Salary Status Classification
* Automated Excel Calculations

## Dataset

| Employee Name | Basic Salary |
| ------------- | -----------: |
| Rahul         |        20000 |
| Priya         |        25000 |
| Ravi          |        15000 |
| Sneha         |        30000 |
| Kiran         |        18000 |

## Formulas Used

### Bonus Calculation

=IF(B2>=20000,B2*10%,B2*5%)

### Total Salary

=B2+C2

### Salary Status

=IF(D2>=20000,"High Salary","Low Salary")
